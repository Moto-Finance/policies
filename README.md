# Policies

Public-facing policies for **Moto**, the operating company behind [moto-card.com](https://moto-card.com).

This repo is the canonical source for our public policies. It exists to give customers, partners, security researchers, and regulators a single, version-controlled reference for how we operate. All material changes are visible in Git history.

## Contents

- [`SECURITY.md`](./SECURITY.md) — Vulnerability disclosure policy and how to report security issues

## Reporting a security issue

Please follow [`SECURITY.md`](./SECURITY.md). For sensitive reports, email `security@moto-card.com` (PGP key fingerprint published in the policy). Do not file public issues in this repo for security matters.

## Changes

Material changes are reflected at the top of each policy with an effective date and, where required, communicated directly to affected users. Pull requests from the public are not accepted.

## Contact

- Security: `security@moto-card.com`
- Everything else: `legal@moto-card.com`
